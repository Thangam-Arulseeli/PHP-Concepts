AJAX - INTRODUCTION 

 
    
    *** AJAX stands for Asynchronous JavaScript and XML.
      *  AJAX is a new technique for creating better, faster, and more interactive web applications
         with the help of XML, HTML, CSS and Java Script.
      * Conventional web application transmit information to and from the sever 
         using synchronous requests. This means you fill out a form, hit submit, 
         and get directed to a new page with new information from the server.
      * With AJAX when submit is pressed, JavaScript will make a request to the server,
         interpret the results and update the current screen. 
         In the purest sense, the user would never know that anything was even transmitted 
         to the server.
