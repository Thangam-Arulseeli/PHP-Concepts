        *** Important Error:
        Warning: mail(): Failed to connect to mailserver at "localhost" port 25, verify your "SMTP" and "smtp_port" setting in php.ini or use ini_set() in D:\xampp\htdocs\PHP-Concepts\mail\sendmail.php on line 19
        Mail could not be sent!!!
        *** To rectify above error [Configure php.ini file] 
        1. Go to d:/xampp/php/
        2. Search for smtp [It is in mail function] 
        3. Get the following smtp details 
            i. SMTP=localhost
            ii. smtp_port=25
            iii. sendmail_from=
            iv. sendmail_path=
            -- ----------------------------
          a. Change the details  [Search smtp]
          [mail function]
          ; For Win32 only.
          ; http://php.net/smtp
          SMTP=localhost
          ; http://php.net/smtp-port
          smtp_port=25
          -- ------------------------------------
          [mail function]
                ; For Win32 only.
                ; http://php.net/smtp
                SMTP=smtp.gmail.com     // Change it
                ; http://php.net/smtp-port
                smtp_port=587    // Change it

                ; For Win32 only.
                ; http://php.net/sendmail-from
                sendmail_from=seeliseelan@gmail.com    // Change it

                ; For Unix only.  You may supply arguments as well (default: "sendmail -t -i").
                ; http://php.net/sendmail-path
                sendmail_path = "\"D:\xampp\sendmail\sendmail.exe" -t"     // Change it

          -- ----------------------------------
        4. Go to sendmail.ini file in d:/xampp/sendmail folder and enter the follwing 
                [sendmail]
        
                smtp_server=smtp.gmail.com
                smtp_port=587
                error_logfile=error.log
                debug_logfile=debug.log
                auth_username=seeliseelan@gmail.com
                auth_password=my-gmail-app-password
                force_sender=seeliseelan@gmail.com
                
            IMPORTANT NOTE:: To generate my-gmail-app-password
